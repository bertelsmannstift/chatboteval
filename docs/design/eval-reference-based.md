# Reference-Based Evaluation Design

## Overview

Evaluation layer that scores chatbot responses against human-annotated ground truth. Trains supervised models (DeBERTa multilabel classification, SBERT similarity, cross-encoder reranking) on annotated data and uses them for inference on new query-response pairs.
